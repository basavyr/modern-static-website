Explore & analyze logs using Elasticsearch

A fully working server which ingests logs (e.g. syslog, system metrics etc.) from any local or external sources. Ingested logs are parsed with Logstash and then the output is sent to Elasticsearch. Log analysis is done through Kibana UI.

Read More

Elasticsearch-Logstash-Kibana full stack service developed on a dedicated server.

  • The entire stack works on a CentOS 7 virtual machine, inside the DFCTI data center.
  • Communication between each service inside the ELK stack is encrypted with TLS.
  • A static web server is deployed through NGINX, which allows external access to the main website dedicated to this project.
  • Kibana UI accessible from anywhere through NGINX reverse proxy.
  • Renewable certificate of encryption is configured on the virtual machine through certbot.
  • Daily maintenance for keeping the ELK stack up-to-date.

Elasticsearch

Distributed, RESTful search and analytics engine capable of addressing multiple use cases.

Logstash

Data processing pipeline that ingests & transforms data from a multitude of sources then outputs it to ES.

Kibana

Free and open user interface that allows to visualize Elasticsearch data and navigate the Elastic Stack.
