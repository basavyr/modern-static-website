The current ELK platform

Currently, there are two main development pipelines which involve log analysis. Each pipeline is a fully fledged ELK stack that runs on elk.nipne.ro VM, ingesting logs from different sources, both with completely different log formats, then the parsed data are stored into separate indexes. With Kibana, visualizing & interpreting the data from each index is straightforward.

x

Reverse proxy with NGINX: allows one to remotely connect to Kibana UI and the ES indices from the browser. Fully supported queries are available in the browser for Kibana and ES.
